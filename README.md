# Laravel-Forms

## Summary
Online form creation and fill support implemented in Laravel.\
The app supports:
- Registering and logging in
- Creating new forms, with dynamic content, that support textboxes, single and multiple chioce options
- Editing existing forms if they are not already filled
- Filling forms
- Visualising the data of filled forms
- Soft deletion of forms which are not filled yet

## How to Run
```
1. Install Node, PHP and Composer
2. Run init.bat or init.sh depending on your system.
```
